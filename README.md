# WC-NG6
Create a WebComponent with Angular6

## Prerrequisitos
- Tener instalada la ultima versión de angular cli
npm i -g @angular/cli


## Creacion del proyecto

ng new bz-card --prefix custom


## Ejecutar
ng serve

Esto nos permitira verificar la aplicación en http://localhost:4200
